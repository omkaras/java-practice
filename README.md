# Java Practice 🚀

A curated collection of Java programs, concepts, and problem-solving exercises designed for **interview preparation** and **core Java mastery**.

This repository focuses on building strong fundamentals in:

* Core Java
* Data Structures & Algorithms
* Concurrency & Multithreading
* System Design concepts (Java-centric)

---

## 🧠 Topics Covered

### 🔹 Core Java

* JDK, JVM
* OOP (Encapsulation, Inheritance, Polymorphism)
* Interfaces, Sealed Classes, Records
* Equals & HashCode
* Generics
* Reflection
* Immutable & Singleton classes

### 🔹 Concurrency

* Threads & ExecutorService
* Virtual Threads
* CompletableFuture & Async
* Volatile & CAS
* Synchronization vs Locks

### 🔹 Collections & Internals

* HashMap & ConcurrentHashMap
* Hashing concepts
* Memory management & Garbage Collection

### 🔹 Frameworks

* Spring Dependency Injection
* Spring Autoconfiguration
* @Transactional
* Hibernate & JPA (N+1 problem)

### 🔹 Data Structures

* Stack, Queue
* Linked List (Singly & Doubly)
* Trees
* LRU Cache

### 🔹 System Design

* Microservices
* API Gateway & Service Mesh
* Kafka, RabbitMQ
* CAP Theorem
* Caching & Load Balancing
* Backpressure handling

---

## 💻 Sample Problems

* Maximum Subarray Sum (Kadane’s Algorithm)
* Linked List Intersection
* Middle of Linked List
* Sort 0s, 1s, 2s (Dutch Flag)
* String Compression
